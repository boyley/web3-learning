10 · Etherscan 区块浏览器 · 链接生成 + 对照学习

本工具帮你把「交易哈希 / 地址」变成可点击的 Etherscan 链接，并对照说明页面上每一栏对应链上的什么概念。 纯前端、无需联网安装、不需要私钥。

③ 交易详情页 · 栏目对照表

Etherscan 栏目对应链上概念（见对应模块）
Transaction Hash交易唯一 ID（03 交易）
Status成功/失败 = 收据 status（03）
Block / Confirmations所在区块 + 确认数（06 区块）
From / To发送方 / 接收方，To 是合约会标名（02 账户）
Value转账 ETH 数量（08 单位）
Transaction Fee手续费 = gasUsed × 单价（04 Gas）
Gas Price / Base / PriorityEIP-1559 单价拆解（04 Gas）
Nonce发送方交易序号（02/03）
Input Data调用合约的 data：函数选择器+参数（09 RPC）
Logs / Tokens Transferred合约发出的事件日志
⚠️ 安全提示：只用官方域名 etherscan.io / sepolia.etherscan.io。 区块浏览器只是「看数据」，正常浏览绝不需要你签名或连接钱包授权——遇到要你签名的「浏览器」就是钓鱼。 合约显示 Verified 只代表源码与字节码一致，不代表安全；假币可能冒用知名代币名，务必核对合约地址。
